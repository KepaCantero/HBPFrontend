The Neurorobotics platform

The NeuroRobotics Platform (NRP) developed in the Human Brain Project (HBP) is an Internet-accessible simulation system that allows the simulation of robots controlled by spiking neural networks.

Targeted Audience

The NRP platform targets researchers of multiple fields. Prospected users include but are not limited to neuroscientists wanting to validate brain models in the context of closed action-perception loops as well as robotics researchers wanting to develop new neuro-inspired controllers.

What to expect?

This is the first internal release of the NRP, targeted at the HBP consortium. It offers a limited set of functions - more features will follow in regular intervals.

The first initial release demonstrates the integration of robot and environment simulations (the World Simulation Engine) with a brain simulation (HBP Netsim aka NEST).

Future releases will offer graphical interfaces to select and configure robot and environment models as well as programmable interfaces to connect brain models to the sensors and effectors of a robot. An experiment designer will help users to define their own experiment protocols.

We illustrate the current state of the NRP with a simple experiment: a mobile robot that moves inside a realistically modelled virtual room with two computer screens. The user can interact with the experiment by changing the colors of the screens. The robot tries to reach the red screen by virtue of a small network of spiking neurons, turning the robot into a Braitenberg vehicle.

We provide two versions of this experiment. The first uses a four-wheeled mobile robot (Husky) and the second experiment uses a six-legged walking machine (LAURON V).

From your browser!

You access the platform from your browser (see list of supported browsers). When you log into the platform you can start a new experiment or join other experiments that are currently running. If you start a new experiment, you can change the experiment (currently just some parameters) and interact with the simulation. If you join a running experiment, you are only an observer, but you can still navigate through the scene.

The platform should be easily accessible to researchers from neurosciences and robotics alike with little to no programming involved for setting up and running experiments. In later releases, the NRP will provide an easy way to use high-performance computing clusters as well as neuromorphic hardware from the respective HBP subprojects.

Your role in the NRP

We are looking for scientists interested in doing in silico cognitive and behavioral neuroscience experiments with the NRP as part of a co-design project. These co-design projects will drive NRP development (as well as other platforms) from the user perspective by ensuring that all new features and capabilities are to the benefit of the platform users. For application, please use the survey or contact the NRP team directly by email.

Your feedback is welcome. In order to help us improve the NRP you can answer this survey and report issues by filling out a bug report . In addition, the mailing list can be used to discuss further details of the software. There, we would like to discuss aspects of further developments, e.g., what kind of experiments should be conducted and what requirements arise.

Roadmap

The main focus of NRP's ongoing and future development is the implementation of user tools for designing custom experiments. This will enable experimenters to comfortably specify more sophisticated experiments with well-defined control flows, to select (and, to a certain extent, to create) virtual robots and environments, and to configure the connections between brain models and the embodiments.
