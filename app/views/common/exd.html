Experiment Designer

The Experiment Designer development will start soon.

The Experiment Designer will offer a graphical interface to design a whole experiment:

  • choose a robot
  • choose an environment
  • choose a brain
  • choose/edit a bibi brain-robot mapping scheme
  • define start and stop conditions for experiment as well as timeout and loop cycle times
  • create events, define user interaction possibilities
  • design a time sequence diagram and plan experiment

Meanwhile...

The existing experiments have been predefined by us. The brain, robot, environment have been put together statically in the experiment description files on our servers. Still, if you go to the Experiment Simulation Viewer page, you will be presented a list of predefined experiments, which means that you have the choice among different combinations of brains/robots/environements.

In the near future, you will be able to choose this combination by yourself through a page that should resemble the one hereunder.
